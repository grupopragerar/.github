# Repertório 2022 | Pra Gerar
 
#### Legenda Campo Hamônico
* `T` = Tom 
* `N` = Nota/Acorde

> `T (N)` = Inicia na nota `N`, do Campo Harmônico de `T`

> `T / T2` > Tom `T` modula pra tom 2 `T2` após a barra `/`
